# Lamia Documentation

Lamia is a human-friendly scripting language with seamless Python integration and a comprehensive automation and validation framework that provides powerful tools for data validation, web automation, and LLM integration. It is designed for non-technical users.

## Quick Start

Get started with Lamia in just a few steps:

1. **Installation**
   ```bash
   pip install lamia
   ```
